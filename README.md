# merge_sort
Implementation for merge sort.

Input: N, and N rows where each row contains an unsigned number.(N <= 500000)

Output: N rows of sorted numbers in ascending order.


Sample Input:

10

498478549

358326845

973168785

811831085

291714546

576912541

243572795

848256633

378382144

102398770


Sample Output:

102398770

243572795

291714546

358326845

378382144

498478549

576912541

811831085

848256633

973168785
